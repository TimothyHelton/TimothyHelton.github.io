Background

Recently I was able to take an introduction to C++ course and very quickly realized there is a ton of power in this language, but you have to implement everything. At the moment most of my day-to-day programming at work and in the K2 Data Science Bootcamp is done in Python, which has many open source resources to help a programmer be more efficient. Since it appears a large slue of C++ projects is coming down the pipe at work, I needed to get up to speed.

Previously, I set up a bash script to initialize a Python project and decided it would be worth the time to set up a similar C++ Project Template .

The template uses the following technologies configured to work with Clion , an IDE by JetBrains :

  • CMake: Build configuration
  • Doxygen: API documentation
  • Google Test: Unit Testing

Thanks for looking around and enjoy the day!

Quick Start Guide

  1. Download or Clone the C++ Template from GitHub
  2. In the top level CMakeLists.txt file set the project name on line 2 and the version of the C++ compiler on line 4. 
                                    cmake_minimum_required(VERSION 3.9)
                                project(NewProject)
                                
                                set(CMAKE_CXX_STANDARD 17)
  3. Update the following variables in the Doxygen configuration file docs/Doxyfile. 
                                    PROJECT_NAME = "NewProject"
                                PROJECT_NUMBER = 0.0.0
                                PROJECT_BRIEF = "Project Statement"
  4. Rebuild the CMake file.
  5. The CLion IDE allows you to mark directories with their desired purpose. To mark a directory right click on the directory name in the Project window and select Mark Directory as from the drop-down menu.
    1. Mark the src directory as Project Sources and Headers
    2. Mark the tests/lib/googletest directory as Library Files
  6. Setup the Run/Debug Configuration by selecting Edit Configurations... from the pull-down menu from the run button (green triangle) in the upper right corner.
    1. Select Doxygen from the Application menu on the left.
    2. Choose the executable for Doxygen to be Unit_Tests_run.
    3. In the upper left corner select the plus symbol.
    4. Chose Google Test from the drop-down menu.
    5. Set Name to Unit Tests.
    6. Set Target to Unit_Tests_run.

That should be all it takes to start writing code. If you find any issues or bugs with this repository please file an issue on GitHub .

Hope you find this template useful and enjoy learning C++!
